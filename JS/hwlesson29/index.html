Як можна використовувати JS?

  • - для повідомлень для юзера
  • - первинна валідація
  • - для відображення контенту без перезагрузки сторінки
  • - анімація
  • - робота з відео
  • - збір даних
  • - розрахунки
Імена змінних:
  • - fullName
  • - nameLastName
  • - nameSurname
  • - nameFamilyName
Види коментування:
- // (строкове)
- /* (багатострокове)

Види написання змінних:
  • - camelCase;
  • - з $ або _;
  • - з цифрами (але не на початку)
  • - верхній та нижній регістр (на початку тільки нижній)
  • - не вживати зарезервовані слова (let, function, class, return)
